# 🎮 Number Guessing Game (Python Beginner Project)

This is a simple Python mini-game I built while learning the basics of programming. The game challenges the player to guess a randomly generated number within a selected difficulty level.

## 🚀 Features

* 🎯 Three difficulty levels:

  * Easy (0–100)
  * Medium (0–500)
  * Hard (0–1000)
* 🔁 Replay option after each game
* 📊 Tracks number of attempts
* 🧠 Provides hints ("Too high" / "Too low")

## 🛠️ Technologies Used

* Python (basic concepts)

  * Loops
  * Functions
  * Conditional statements
  * Random module

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the file:

```bash
python your_file_name.py
```
